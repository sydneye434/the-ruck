# Screenshots

The images referenced from the root **`README.md`** live in this folder.

- **`*.svg`** — stylized **illustrative** mockups (dark theme, pink accents) so the repo stays lightweight and renders on GitHub without checking in large PNGs.
- To use **real captures**: run `npm run dev` + `npm run seed`, take PNG/WebP screenshots, save them here (e.g. `dashboard.png`), and update the paths in **`README.md`**.

Suggested capture sizes: **1200–1600px** wide for clarity; compress PNGs or use WebP for smaller clones.
